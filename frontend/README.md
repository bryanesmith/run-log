# Run Log: frontend

## Overview
Frontend application for logging runs and tracking progress.

## Setup

**Note**: You must start the backend server first.

Copy `src/config-template.json` to `src/config.json` and edit as needed.

Start the development server:

```sh
$ yarn && yarn start
```

To build:

```sh
$ yarn && yarn run build
```

## Tests

```sh
$ yarn test
```
